# CountLines
a tool for counting code lines written by C# 2.0

可以统计任何文本文件类型 ，可以自定义要统计的类型，默认情况下只统计.cs，.Java，.cpp，.c类型的文件,需要.NET2.0框架

主要功能：

统计单个文件代码行数

统计文件夹代码行数

统计磁盘代码行数

可以设置要统计的文件类型

可以自定义统计的文件类型

可以去除“{}”和空行进行统计

可以保存统计结果列表

使用简单方便，容易操作，默认情况下只统计.c，.cpp，.Java，.cs类型的文件，如需统计其它类型文件请先进行设置
